World-Building Codex - Site Plan

Site Name

World-Building Codex

This name reflects the purpose of the site: a structured, encyclopedia‑style reference for a fictional world.

Site Purpose

The website will serve as a compact encyclopedia for a fictional setting. It will organize information into categories such as factions, locations, creatures, artifacts, and cultural concepts. Visitors will be able to explore the world through structured entries and intuitive navigation.

Scenarios
